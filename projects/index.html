HTML5 Audio Tag!

This is my second class project to check out my other projects, please follow the links at the bottom of the page.

At this time i want to share with all of you some of my favorite stuffs which will work for all browsers, i will create a simple HTML5 audio player.

If you need help creating your own audio player page please do not hesitate to contact me, i will be very happy to help.

Some of the browsers do not support the same audio file, so we have to work on different file formats for the audio element by using mp3, wav, ogg.

Here is the sample code of HTML5 audio tag:

      
      <audio controls autoplay>
        source src="music.ogg" type="audio/ogg"
        source src="music.mp3" type="audio/mpeg"
        source src="music.wav" type="audio/wav"
      </audio>

I have linked three audio types as mp3 wav and ogg, so each browser can automatically find the supported file.

Please see the table below showing browsers supporting HTML5 audio, i have already tested on my browsers 6 media types, as mp3, ogg, wav, m3u, aac and pls.

Note: m3u aac and pls are formats playing continuously audio stream.
